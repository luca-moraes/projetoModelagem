Enviar ordem de venda

UMLUseCase
Untitled :: Model :: Enviar ordem de venda

Description

Fluxo principal:

  1. Usuário acessa a área de novas ordens de venda de ações.

  2. Usuário escolhe a ação desejada para efetuar uma nova ordem de venda, podendo utilizar filtros para as opções exibidas.

  3. Usuário escolhe o tipo de ordem de venda, sendo eles: OV com gatilho, OV a mercado, OV por quantidade de ações e OV por valor financeiro.

  4. Usuário preenche e confirma as informações da OV.

  5. Ordem de venda é postada pela corretora.

Fluxo alternativo:

  1. Ordem de venda não é efetuada pois, seu valor está abaixo da cota mínima de venda exigida pela corretora, usuário é avisado que necessita vender uma quantidade maior de ações para postar aquela ordem.

Requisitos referentes ao caso de uso:

  • 17, 21, 23 e 25.
