Cancelar ordem de venda

UMLUseCase
Untitled :: Model :: Cancelar ordem de venda

Description

Fluxo principal:

  1. Usuário acessa a área de ordens de venda postadas.

  2. Usuário escolhe a ordem de venda desejada, podendo utilizar filtros para as opções exibidas.

  3. Usuário escolhe e confirma a opção de cancelamento de OV.

  4. Corretora retira OV das ordens postadas.

Fluxo alternativo:

  1. O cancelamento não é efetuado pois a ordem já foi confirmada como realizada pela empresa, usuário é avisado que a ordem não pode ser cancelada.
