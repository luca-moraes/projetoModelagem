Modificar ordem de venda

UMLUseCase
Untitled :: Model :: Modificar ordem de venda

Description

Fluxo principal:

  1. Usuário acessa a área de ordens de compra postadas e entra na opção de ordens não efetuadas.

  2. Usuário escolhe a ordem de compra desejada, podendo utilizar filtros para as opções exibidas.

  3. Usuário escolhe e confirma a opção de modificar OC.

  4. Usuário ajustar as informações da OC e confirma modificação.

  5. Corretora modifica as informações da OC nas ordens postadas.

Fluxo alternativo:

  1. A modificação não é realizada pois a ordem acabou de ser confirmada como efetuada pela empresa no momento da alteração, o usuário é avisado que a ordem não pode ser modificada.

Requisitos referentes ao caso de uso:

  • 10.
